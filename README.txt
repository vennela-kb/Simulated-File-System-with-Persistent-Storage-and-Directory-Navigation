Commands: 

1. mkdir <dirname>:Creates a new directory with the specified name.

	Example: mkdir testdir

2. cd <dirname> : Changes the current working directory to the specified directory.

	Example: cd testdir

Special case: cd .. will move you up one directory level (parent directory).

3. ls: Lists all files and directories in the current directory.

	Example: ls

4. pwd: Prints the current working directory (path).

	Example: pwd

5. create <filename>: Creates a new file with the specified filename. After running this command, you'll be prompted to enter data for the file.

	Example: create testfile.txt

After entering the command, it will ask you for data:

Enter data to write to testfile.txt: (Write any text when it ask you to put content).

6. read <filename>: Reads and displays the contents of the specified file.

	Example: read testfile.txt

7. delete <filename>: Deletes the specified file.

	Example: delete testfile.txt

8. help: Displays a list of available commands and their descriptions.

	Example: help

9. exit: Exits the shell and stops the program.

	Example: exit
